Graphite Alerts
===============

Using ASP.NET MVC4.

Features
--------
- Define alerts based on graphite monitors.
- Display alerts in web page, with critical on top.
- Read alert definitions from configurable folders.
- Refresh of all alerts, using ajax refresh.
